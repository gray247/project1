# Development Notes

## Running the app
- Install dependencies: 
pm install
- Start the Electron app: 
pm start

## Browser extension
- With the Electron app running, use the browser extension to select ChatGPT messages via injected checkboxes.
- The extension sends selected messages to the local SnipBoard backend at http://127.0.0.1:4050/add-clip.
